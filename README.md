# Pygame-renderer-test
Simple Pygame diagnostic tool for testing rendering, input handling, and SDL functionality.

## Requirements

- Python 3.8 or newer
- Pygame 2.0 or newer

## Installation

Clone the repository:

git clone https://github.com/Frank16243/Pygame-render-test.git

Install dependencies:

pip install -r requirements.txt

## Usage

python pygame_test.py

## Expected Result

A window titled "Pygame Render Test" should appear displaying:
- A rotating rectangle
- Two circles
- A status message

Press ESC or close the window to exit.
